# Quotes on Dev

A site that cycles through an archive of quotes from developers on button click.

## Installation

1. Clone this project and add to your htdocs in MAMP.

2. Run gulp.


# Technologies

* HTML/php

* CSS/sass

* JS/Gulp

* Wordpress Suite
